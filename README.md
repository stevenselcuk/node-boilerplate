## Features

- [x] Babel 7
- [x] Webpack
- [x] Eslint 
- [x] Prettier
- [ ] Test
- [ ] Test Coverage
- [ ] CI
- [ ] Github Actions

## Start it!

`git clone https://github.com/stevenselcuk/node-boilerplate yourawesomeproject`

`cd yourawesomeproject`

`npm install`

`npm run dev`
